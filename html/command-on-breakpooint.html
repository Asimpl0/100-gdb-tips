给断点添加命令

例子

```c

include

int func(int a) { return (++a); }

int main(void) { printf("%d\n", func(1)); return 0; } ```

技巧

在 gdb 中，可以通过"commands [bnum]"命令来给指定的断点添加命令，在该断点触发后即会自动执行该命令，bum 可以为单个断点的编号如 5，也可以为范围如 5-9，不指定 bum 时，则给最后一个设置的断点添加命令。以上述程序为例：
